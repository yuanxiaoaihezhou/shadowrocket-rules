# Shadowrocket Rules & Modules

本仓库是对 [yfamilys.com](https://yfamilys.com) 提供的 Shadowrocket 规则和模块的备份与审核。每日通过 GitHub Actions 自动检查上游更新，有变化时创建 Pull Request 供人工审核，确认安全后合并生效。

## 使用方法

在 Shadowrocket 中直接填入下方的订阅地址即可使用本仓库托管的规则或模块。
